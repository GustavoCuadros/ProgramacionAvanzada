Resumen de las primeras dos semanas de Programación Avanzada

-Se comenzó a revisar el concepto de "Clases" utilizando el entorno de DevC++ en
	el cuál se realizó un programa para escribir una nota a un destinatario.
-Se modificó el programa anterior de tal manera que se crearan "Objetos" y se
	mostró el funcionamiento de estos. Se vio la interacción entre los distintos
	archivos.
-Para el tema de "Métodos" se utilizó programación en JAVA. Además se explicó el
	concepto de "superclase" y las principales diferencias entre el lenguaje
	C++ y JAVA. Se realizó un programa que pedía datos al usuario y estos se
	mostraban en la pantalla siguiendo los conceptos anteriormente revisados.
